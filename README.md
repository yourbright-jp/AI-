# 🎙️ AI音声面接システム（AI Voice Interview System）

外国人技能実習生・特定技能の採用面接を自動化するWebアプリケーションです。

## 機能

- **AI音声質問**: AIが日本語で面接質問を読み上げます（Web Speech API）
- **音声認識回答**: 候補者がマイクで回答すると自動的にテキスト化されます
- **全体録音**: 面接全体を録音し、.webmファイルとしてダウンロード可能
- **履歴書読み取り**: 画像/PDF/テキストで履歴書をアップロード → AIが評価に活用
- **AI評価**: Claude APIが5項目でスコアリング・合否判定・文法修正を生成
- **メール送信**: 評価レポートと録音ファイルをメールで送信

## 対応職種

| 職種 | 質問数 | 内容 |
|------|--------|------|
| 🏥 介護 | 17問 | 介護理解・デイサービス・宗教対応など |
| 🏗️ 建設 | 10問 | 体力・高所作業・安全意識など |
| 🍽️ 飲食 | 10問 | 接客・アレルギー対応など |
| ⚙️ 機械 | 10問 | 図面・細かい作業・安全など |
| 📋 その他 | 8問 | 汎用質問 |

## 使い方

### ブラウザで直接使用

1. `index.html` をダウンロード
2. ブラウザ（Chrome/Edge推奨）で開く
3. Anthropic APIキーを入力（初回のみ）
4. 候補者名・職種を選択して面接開始

### Webサイトとして公開

1. `index.html` をNetlifyなどにデプロイ
2. URLを候補者に共有
3. 候補者がアクセスして面接を受ける
4. 結果が指定メールアドレスに送信される

## 技術仕様

- **フロントエンド**: React 18 + Babel（CDN、ビルド不要）
- **音声合成**: Web Speech API (SpeechSynthesis)
- **音声認識**: Web Speech API (SpeechRecognition)
- **録音**: MediaRecorder API → WebM
- **AI評価**: Anthropic Messages API (Claude Sonnet 4)
- **ホスティング**: Netlify（無料プラン）

## AI評価の出力

- **5段階スコア**: 各評価項目を1〜5で採点
- **合否判定**: 合格 / 保留 / 不合格
- **日本語レベル推定**: N5〜N1
- **文法修正**: 間違った表現の修正＋ベトナム語での解説
- **履歴書照合**: アップロードされた場合、職歴との整合性を分析
